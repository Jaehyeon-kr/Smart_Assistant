## 🚀 다음 단계

### 즉시 시작 가능
```bash
# 백엔드 서버 시작
npm run dev
# http://localhost:4000
```

### 주의사항
1. **Claude API 키 설정**
   - .env의 ANTHROPIC_API_KEY를 실제 키로 변경

2. **프론트엔드 설정**
   - 다음: `npx create-react-app lecq-frontend`

3. **테스트**
   - Postman이나 curl로 API 테스트
   - DB 확인: `sqlite3 db/lecq.db`
